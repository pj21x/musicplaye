As Music has become an important part of our life through which we which we enjoy, concentrate or relax our mind. The Project ‘Music Streaming Platform’ allows a single individual to stream songs from any browser of their choice and explore Artists, Songs and other Playlists. Listen continuously without any interruption.


Goal: - Expand the social functionality of a music streaming by enabling music lovers to simultaneously listen to songs without downloading any special required software or application which eliminates storage or system space requirement. It can be access directly with active internet connection through any browser.


This Site is developed as the project for final year of BCCA Semister 6.
College : City Premier College.
University : Rashtrasant Tukadoji Maharaj Nagpur University
Under the Category of Web Based Application


Web application development is usually made on client-server architecture and use a web-browser as the client interface. This is one of the reasons why web applications are so widely getting popular. Though web applications offer a slight advantage over desktop applications, as there is no special need of installing software, there is a very narrow chance of desktop applications becoming outdated.
